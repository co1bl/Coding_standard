Image Annotation Tool

How to use:

1. Click anywhere on the image to place a 100×100 pixel bounding box

2. Choose the appropriate annotation type (A, B, C, D, E, or F) to save the crop

3. Use "Previous" and "Next" to navigate between images

4. Click "Clear Selection" to remove the current bounding box

Fixed Bounding Box Size: 100 × 100 pixels
